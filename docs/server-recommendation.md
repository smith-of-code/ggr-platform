# Рекомендация по выбору серверной инфраструктуры

**Дата:** Февраль 2026 г.
**Подготовлено:** Rakhmanov.studio

## 1. Требования проекта

### 1.1 Технологический стек

| Компонент | Технология | Требования к ресурсам |
|---|---|---|
| Backend | Laravel 12, PHP 8.3 (PHP-FPM) | CPU-интенсивный, RAM ~2–4 ГБ |
| Frontend | Vue 3, Inertia.js, Vite (SSR-ready) | Сборка: RAM ~2 ГБ |
| СУБД | PostgreSQL 16 | RAM ~4–6 ГБ, быстрый I/O |
| Кеш / Очереди / Сессии | Redis | RAM ~1–2 ГБ |
| Web-сервер | Nginx (Alpine) | Минимальные |
| Queue Worker | Laravel Horizon (Supervisor) | CPU + RAM ~512 МБ |
| Контейнеризация | Docker, Docker Compose | Оверхед ~512 МБ |
| Почтовый сервер | Mailpit (dev) / SMTP (prod) | Минимальные |

### 1.2 Функциональные модули платформы

- **LMS**: курсы с мультиблочными этапами, модули, SCORM-пакеты, видео, тесты, задания
- **Управление пользователями**: создание, импорт Excel/CSV, ролевая система, приглашения
- **Геймификация**: баллы, рейтинги, автоматические триггеры
- **Конструктор форм**: опросы, анкеты, embed на внешние сайты
- **Отчёты и аналитика**: статистика обучения, экспорт, email-рассылка
- **SCORM**: загрузка ZIP, распаковка, iframe-отображение (дисковое пространство)
- **Видеоматериалы**: внешние ссылки (Rutube, YouTube) — без локального хранения

### 1.3 Расчёт ресурсов

| Ресурс | Минимум | Рекомендуемый | Обоснование |
|---|---|---|---|
| **CPU** | 4 ядра | **8 ядер** | PHP-FPM worker'ы (4–8), PostgreSQL, Redis, Nginx, cron, queue worker |
| **RAM** | 8 ГБ | **16 ГБ** | PHP-FPM ~4 ГБ, PostgreSQL ~4 ГБ, Redis ~2 ГБ, Nginx + Docker ~2 ГБ, запас ~4 ГБ |

| **Сеть** | 200 Мбит/с | **1 Гбит/с** | Одновременная работа 100+ участников, загрузка SCORM/файлов |

## 3. Преимущества Timeweb Cloud

### 3.1 Ценовое преимущество

- **Лучшая цена/производительность** среди российских IaaS-провайдеров (подтверждено рейтингами)
- Диск NVMe, DDoS-защита, 1 Гбит/с канал **включены** в стоимость — нет скрытых платежей
- Скидка **10% при оплате за 12 месяцев** → 58 752 ₽/год вместо 65 280 ₽

### 3.2 Производительность

- Процессоры **3.3 ГГц** — высокая частота, критична для PHP-FPM и PostgreSQL
- **NVMe SSD** — низкая задержка I/O, ускоряет запросы к БД и загрузку SCORM
- Сетевой канал **1 Гбит/с** — достаточно для одновременной работы 200+ пользователей

### 3.3 Инфраструктурные преимущества

- **Дата-центры Tier III** в России (Москва, СПб, Новосибирск)
- Собственная панель управления — удобная и быстрая
- **API, Terraform, CLI, Cloud-init** — полная автоматизация развёртывания
- Возможность масштабирования «на лету» без миграции
- **Лауреат Премии Рунета** 2025 в номинации «Облачное решение года»
- **132 000+** реальных клиентов

### 3.4 Соответствие требованиям

- Включён в **Единый реестр российских программ** (запись №15725)
- Соответствие **152-ФЗ** (обработка персональных данных)
- Подтверждён **высший уровень защищённости** персональных данных (январь 2026)

### 3.5 Поддержка

- **24/7** поддержка: чат, мессенджеры, тикеты, телефон
- Помощь с миграцией и администрированием
- Документация и комьюнити-чат

### 4.2 Развёртывание на сервере

```
Timeweb Cloud MSK 160 (8 vCPU / 16 ГБ / 160 ГБ NVMe)
├── Docker Engine + Docker Compose
│   ├── nginx (reverse proxy, SSL)         ~100 МБ RAM
│   ├── php-fpm (Laravel 12, PHP 8.3)     ~4 ГБ RAM (8 workers)
│   ├── postgres:16                        ~4 ГБ RAM
│   ├── redis:alpine                       ~1 ГБ RAM
│   ├── laravel-horizon (supervisor)       ~512 МБ RAM
│   └── certbot (SSL Let's Encrypt)        периодически
├── Свободно                              ~6 ГБ RAM (запас для пиков)
└── Диск
    ├── ОС + Docker images                ~15 ГБ
    ├── PostgreSQL data                   ~20 ГБ
    ├── SCORM-пакеты                      ~30 ГБ
    ├── Uploads (задания, файлы)          ~20 ГБ
    ├── Логи                              ~10 ГБ
    └── Запас                             ~65 ГБ
```

### 4.3 Рекомендации по безопасности

- Настройка firewall (UFW): открыть только порты 22, 80, 443
- SSL-сертификат через Let's Encrypt (certbot)
- Регулярные бэкапы БД (pg_dump → S3-хранилище Timeweb)
- Обновление ОС и Docker-образов
- Fail2ban для защиты SSH

### 4.4 Масштабирование (при росте нагрузки)

| Этап | Действие | Стоимость |
|---|---|---|
| Текущий | Cloud MSK 160 (8/16/160) | 5 440 ₽/мес |
