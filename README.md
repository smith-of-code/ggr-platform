# Гостеприимные города Росатома — rosatom-travel.ru

Цифровая экосистема для развития туристического, образовательного и предпринимательского потенциала атомных городов Госкорпорации «Росатом».

## Структура проекта

- `app/` — Laravel-приложение (backend + frontend)
- `docker/` — конфигурация Nginx для Docker
- `docker-compose.yml` — оркестрация контейнеров
- `КП_rosatom-travel_План_разработки.md` — коммерческое предложение с планом и сроками
- `tz.txt` — техническое задание

## Быстрый старт

```bash
cd app
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
npm run build
php artisan serve
```

Откройте http://localhost:8000

## MVP — реализовано

- Главная страница
- Каталог городов и карточка города
- Каталог туров с фильтрами и карточка тура
- Форма заявки на тур
- Админ-панель: дашборд, список заявок
