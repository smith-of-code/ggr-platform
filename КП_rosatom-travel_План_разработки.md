## 1. РЕЗЮМЕ ПРОЕКТА

Сайт rosatom-travel.ru — цифровая экосистема для развития туристического, образовательного и предпринимательского потенциала атомных городов Госкорпорации «Росатом». Цель доработки — трансформация из информационного ресурса в функциональную платформу с личными кабинетами, взаимосвязанными сущностями (города, туры, вакансии, люди) и развитой административной частью.

**Рекомендуемый технологический стек:**
- **Backend:** Laravel 11+ (PHP 8.3+)
- **Frontend:** Vue 3 + Composition API + Pinia
- **База данных:** PostgreSQL 16+
- **Инфраструктура:** Docker / Docker Compose, возможность развёртывания в Kubernetes
- **Кэширование:** Redis
- **Поиск:** Laravel Scout + PostgreSQL Full-Text (или Meilisearch при необходимости)
- **Очереди:** Redis Queue для фоновых задач

## 4. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### 4.1. Архитектура

- Модульная архитектура, API-first
- Разделение frontend (Vue SPA) и backend (Laravel API)
- Масштабируемость (горизонтальное масштабирование приложений)
- Документированный API (OpenAPI 3.0)

### 4.2. Безопасность

- Соответствие 152-ФЗ «О персональных данных»
- Хранение ПДн на территории РФ
- Ролевая модель доступа (RBAC)
- Логирование действий администраторов и модераторов
- Защита от OWASP Top 10 (SQL injection, XSS, CSRF и др.)

### 4.3. Инфраструктура

- Docker-контейнеры (Laravel, Vue, Nginx, PostgreSQL, Redis)
- Возможность размещения на инфраструктуре Заказчика
- Резервное копирование БД
- Мониторинг (логи, метрики, алерты)

### 4.4. Фирменный стиль

- Цвета: Pantone 7687 C (RGB 0 50 116), Pantone 660 C (RGB 2 94 161), Pantone 284 C (RGB 108 172 228), Pantone 7409 C (RGB 252 195 11) и др. по руководству
- Шрифт: Rosatom (Regular, Italic, Bold, Light) или Tilda Sans Bold для веб
- Логотипы из папки 2024_01_Rosatom_GG_logo_output_editable_Folder
- Соблюдение охранных полей, запретов на деформацию и изменение логотипа
