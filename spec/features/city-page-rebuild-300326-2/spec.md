# Герб города в баннере

## Цель

Добавить маленькое изображение герба города в hero-баннер страницы города; в админке — возможность загрузить герб.

## In-scope

- Новое поле `coat_of_arms` (nullable text) в таблице `cities`
- Добавить `coat_of_arms` в `$fillable` модели `City`
- Валидация и сохранение поля в `Admin\CityController` (`store`, `update`)
- Поле загрузки герба в админ-форме (`Admin/Cities/Form.vue`) через существующий `ImageUploadCrop`
- Отображение герба в hero-секции `Cities/Show.vue`:
  - Если картинка JPEG (есть фон) — отображать с декоративной рамкой / окантовкой (белая обводка, тень, скруглённый контейнер)
  - Если картинка PNG/WebP (потенциально без фона) — отображать «как есть», без дополнительной рамки
  - Размер ~64–80 px, позиция — рядом с названием города (слева от `h1`)
- Адаптивность: герб уменьшается на мобильных

## Out-of-scope

- Автоматическое определение прозрачности на сервере (определение по расширению файла на фронте)
- Кроп герба (не нужен, используем стандартный `ImageUploadCrop` без обрезки)
- Каталог городов (`Cities/Index`) — герб не показываем

## Constraints

- Паттерн загрузки — через `route('admin.upload.image')` + URL в текстовом поле (как `image`)
- Миграция — отдельный файл, `after('image')`
- Максимум 5 файлов за шаг, до ~150 строк diff
