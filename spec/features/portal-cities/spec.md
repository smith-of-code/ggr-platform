# Модуль 3. Атомные города

## Статус: Реализован

## Описание
Каталог и детальные страницы атомных городов с расширенным контентом, избранным, инфраструктурой, галереей и связями.

## Каталог городов (Cities/Index)
- Сетка карточек с изображениями
- Переключение вид сетка/список (в планах)
- Поиск и фильтрация (в планах)

## Детальная страница города (Cities/Show)

### Секции
1. **Hero** — полноширинное изображение, название, регион
2. **Кнопки действий** — «В избранное» (для авторизованных), «Поделиться»
3. **Статистика** — население, достопримечательности, туры
4. **О городе** — HTML-описание, интересные факты (chips)
5. **Инфраструктура** — карточки/прогресс-бары из JSON
6. **Достопримечательности** — сетка карточек (title, description, image)
7. **Социальная сфера** — 3 колонки: образование, медицина, культура
8. **Галерея** — сетка фотографий с lightbox
9. **Видео** — iframe YouTube/Rutube
10. **Исследования** — карточки связанных исследований
11. **Рецепты** — карточки связанных рецептов
12. **Туры** — карточки туров в этом городе

### Избранное
- Авторизованные пользователи могут добавлять города в избранное
- POST `/favorites/city/{id}` (toggle)
- Страница всех избранных: `/favorites`

## Модель City — расширенные поля
- `region` — регион
- `population` — население
- `lat`, `lng` — координаты для карты
- `attractions` — JSON массив достопримечательностей
- `social_objects` — JSON {education, medicine, culture}
- `gallery` — JSON массив URL изображений
- `video_url` — ссылка на видео
- `facts` — JSON массив интересных фактов

## Маршруты
- `GET /cities` — каталог (публичный)
- `GET /cities/{slug}` — детальная страница (публичный)
- `POST /favorites/city/{id}` — toggle избранного (auth)
- `GET /favorites` — страница избранных (auth)
