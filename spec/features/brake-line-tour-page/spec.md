# brake-line-tour-page

## Цель

Сохранять переносы строк в описании дней программы тура на публичной странице `/tours/{slug}`.

## В скоупе

- Добавить CSS `whitespace-pre-line` к элементу `<p>` с `day.description` в `Tours/Show.vue`
- Убедиться, что текст, введённый с переносами строк в textarea админки, отображается абзацами на витрине

## Вне скоупа

- Изменение формата хранения `program_days` (JSON остаётся как есть)
- Подключение rich-text редактора для описания дней программы
- Изменение админской формы

## Ограничения

- Минимальное изменение: только 1 строка в `Tours/Show.vue`
- Поведение в админке не меняется

## Открытые вопросы

Нет.
