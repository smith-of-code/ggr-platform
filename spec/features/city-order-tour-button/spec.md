# city-order-tour-button — Кнопка «Заказать тур» на странице города

## Цель

Дать администраторам возможность включить для каждого города кнопку «Заказать тур», при нажатии которой открывается форма заявки (аналог формы «Оставить заявку» со страницы тура).

## В области действия

- Миграция: добавить `show_order_tour_button` boolean (default `false`) в таблицу `cities`.
- `City` model: добавить поле в `fillable` и `casts`.
- Admin `CityController` (store/update): добавить поле в валидацию.
- `Admin/Cities/Form.vue`: добавить чекбокс «Показывать кнопку "Заказать тур"» в правую колонку (рядом с `is_active`).
- `Cities/Show.vue` (клиент): если `city.show_order_tour_button === true` — показывать кнопку «Заказать тур» в hero-блоке (кнопка заметна на фоне, акцентного цвета).
- `Cities/Show.vue`: при нажатии на кнопку открывать модальное окно с формой (title «Заявка на тур»), поля: Имя, Email, Телефон, Сообщение, Согласие на ОПД.
- Форма отправляет `POST /applications` с `type: 'tour'`, `tour_id: null`, `message` включает пометку города.

## Вне области действия

- Изменение логики `ApplicationController::store` (принимает `tour_id: null` уже сейчас).
- Массовое включение кнопки для всех городов.
- Уведомления по email/Telegram (вне этой задачи).

## Ограничения

- `show_order_tour_button` по умолчанию `false` (кнопка скрыта).
- Форма идентична туровской (те же поля), только title «Заявка на тур».
- Используем существующий `applications.store` endpoint без изменений.
- Компоненты: `RModal`, `RInput`, `RButton`, `RCheckbox` из UI-кита.
