# Visible Manager — Управление видимостью страниц портала

## Цель

Дать администратору возможность «скрыть» любую публичную страницу портала: при переходе по URL вместо контента показывается красивая заглушка «Страница в разработке», а ссылка в навигации скрывается.

## В scope

- Админ-интерфейс (раздел «Настройки → Видимость страниц») со списком публичных страниц портала
- Для каждой страницы — toggle on/off (видна / скрыта)
- При переходе на скрытую страницу — Inertia-страница-заглушка с красивым сообщением «Страница в разработке»
- Скрытие ссылок на скрытую страницу из навигации `MainLayout.vue` (header + footer + мобильное меню)
- Хранение настроек в существующей таблице `settings` (group = `page_visibility`)
- Кэширование через `SettingsService`

## Вне scope

- Скрытие страниц LMS / LMS Admin
- Per-user или per-role видимость (только глобально для всех посетителей)
- Кастомные тексты/изображения заглушки (MVP — единый шаблон)
- Скрытие admin-разделов
- SEO-редиректы (503 / noindex) — пока просто 200 с заглушкой

## Ограничения

- Используем существующую таблицу `settings` с group `page_visibility`, key = slug страницы, value = JSON `{"hidden": true}`
- Middleware проверяет видимость на уровне Laravel перед Inertia-рендером
- Список страниц для управления определяется конфигурацией (массив), а не динамически
- Админ видит все страницы без заглушки (middleware пропускает auth-пользователей с admin-доступом)
