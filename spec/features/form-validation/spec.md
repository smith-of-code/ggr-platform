# Фича: Валидация полей форм (пресеты для популярных форматов)

## Goal

Добавить в конструктор форм `/lms-admin/{event}/forms` возможность выбирать у текстовых полей тип валидации (СНИЛС, серия/номер паспорта, ИНН, ОГРН, КПП, дата рождения и т.д.) с применением как на клиенте, так и на сервере.

## In-scope

- Новая колонка `validation` (string, nullable, ≤50 символов) в таблице `lms_form_fields`.
- Каталог пресетов валидации в PHP (`App\Services\Lms\Forms\FieldValidationPresets`) и зеркальный список на фронте (`resources/js/constants/formFieldValidations.js`).
- В конструкторе форм (`Pages/Lms/Admin/Forms/Form.vue`) — селектор «Тип валидации» рядом с типом поля (видим только для `text`/`textarea`).
- Серверная валидация в `FormPublicController::buildPublicFormAnswerRules` + русскоязычные сообщения в `publicFormValidationMessages`.
- Клиентская предварительная валидация в `FormRenderer.vue` и в `public/js/form-widget.js`.
- Прокидывание `validation` в публичный payload (`show`/`apiShow`) и админский (`edit`).

## Out-of-scope

- Произвольные пользовательские regex (только предустановленные пресеты).
- Изменение/добавление новых типов полей (`type`).
- Изменение страницы статистики и логики создания пользователей из ответов.
- Маски ввода (input masks) — в текущей итерации только проверка значения.

## Constraints

- Сохранить существующую авто-детекцию email/phone в `fieldIsEmail`/`fieldIsPhone` (не ломать обратную совместимость).
- Использовать существующие компоненты UI Kit (`RInput`, `RCheckbox`, `RButton`, `RCard`); для селектора оставить нативный `<select>` — единый стиль с селектором типа поля.
- Все новые тексты в UI и сообщения об ошибках — на русском.
- Колонка `validation` обязательна для PHP-пресета (если значение неизвестно — игнорировать).

## Каталог пресетов (первая итерация)

| Ключ | Название (UI) | Формат | Пример |
|---|---|---|---|
| `snils` | СНИЛС | 11 цифр (допускаются `XXX-XXX-XXX YY`), контрольная сумма | `112-233-445 95` |
| `passport_series_rf` | Серия паспорта РФ | ровно 4 цифры | `4509` |
| `passport_number_rf` | Номер паспорта РФ | ровно 6 цифр | `123456` |
| `inn_personal` | ИНН (физ. лицо) | 12 цифр + контрольные суммы | `500100732259` |
| `inn_company` | ИНН (юр. лицо) | 10 цифр + контрольная сумма | `7707083893` |
| `ogrn` | ОГРН | 13 цифр + контрольная сумма | `1027700132195` |
| `ogrnip` | ОГРНИП | 15 цифр + контрольная сумма | `304500116000157` |
| `kpp` | КПП | 9 символов (`NNNN[A-Z0-9]{2}NNN`) | `770701001` |
| `birth_date` | Дата рождения | формат `ДД.ММ.ГГГГ`, не в будущем, возраст ≤ 120 лет | `01.01.2000` |
| `postal_code_rf` | Почтовый индекс РФ | 6 цифр | `101000` |

> Каталог расширяется в одной точке (PHP-сервис + JS-константа); добавление нового пресета не требует миграций.
