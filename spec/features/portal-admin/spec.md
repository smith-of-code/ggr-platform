# Модуль 8. Административная панель портала

## Статус: Реализован

## Описание
Управление всем контентом информационной платформы и модуля «Каталог туров» без участия разработчиков.

## Дашборд
- Общая статистика: города, туры, заявки

## Управление городами
- CRUD с расширенными полями (регион, координаты, достопримечательности, инфраструктура, галерея, видео, факты)
- Toggle активности

## Управление турами
- CRUD с расширенными полями (целевая аудитория, организатор)
- Toggle активности
- Медиа-галерея

## Управление блогом
- CRUD статей с категориями и тегами
- Toggle публикации

## Управление продуктами ВШГР
- CRUD образовательных продуктов

## Управление исследованиями
- CRUD с привязкой к городам

## Управление рецептами
- CRUD с динамическими ингредиентами

## Управление заявками
- Список всех заявок (туры, исследования, обучение, обратная связь)
- Фильтр по типу и статусу
- Экспорт в CSV
- Смена статуса

## Навигация AdminLayout
- Дашборд
- Заявки
- Контент: Города, LMS ВШГР, Туры, Блог
- Образование: Продукты ВШГР
- Исследования: Исследования, Рецепты

## Маршруты
Все под `/admin` с middleware `auth` и `portal.admin` (флаг `users.is_admin`, см. `App\Http\Middleware\EnsurePortalAdmin`):
- `/admin` — дашборд
- `/admin/cities/*` — города
- `/admin/tours/*` — туры
- `/admin/blog/*` — блог
- `/admin/education-products/*` — продукты ВШГР
- `/admin/research/*` — исследования
- `/admin/recipes/*` — рецепты
- `/admin/applications` — заявки
- `/admin/settings/*` — настройки

## Файлы
- `resources/js/Layouts/AdminLayout.vue`
- `resources/js/Pages/Admin/**/*`
- `app/Http/Controllers/Admin/*`
