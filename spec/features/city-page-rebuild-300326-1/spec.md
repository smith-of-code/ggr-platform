# city-page-rebuild-300326-1

## Цель

Заменить блок ключевых метрик (key stats bar) на странице города (`Cities/Show`) — вместо «Население / Достопримечательности / Туры» показывать «Год основания / Население / Часовой пояс» в стиле старого сайта.

## В scope

- Добавить поля `founded_year` (integer, nullable), `population_year` (integer, nullable) и `timezone` (string, nullable) в таблицу `cities`.
- Обновить модель `City` — добавить новые поля в `$fillable`.
- Обновить админ-форму (`Admin/Cities/Form.vue`) — добавить инпуты для `founded_year`, `population_year` и `timezone`.
- Обновить валидацию в `Admin\CityController` для `store` и `update`.
- Переделать блок «Key stats bar» в `Cities/Show.vue`:
  - 3 колонки: год основания, население (с подписью «жителей (по сост. на XXXX год)»), часовой пояс.
  - Стиль: крупные цветные числа (розовый, синий, бирюзовый), подписи снизу, вертикальные разделители.
  - Блок показывается, если хотя бы одно из трёх полей заполнено.
- Убрать `aboutInfographicRows` зависимость от `attractions` / `tours` (эти данные остаются на странице, но не в stats bar).

## Вне scope

- Изменение остальных секций страницы города (About, инфраструктура, галерея, туры и т.д.).
- Изменения на странице списка городов (`Cities/Index`).
- Миграция/заполнение данных для существующих городов (только структура).

## Ограничения

- Дизайн stats bar максимально приближен к скриншоту старого сайта.
- Нет отдельного Node/PHP контейнера — всё через `fpm`.

Нет.
