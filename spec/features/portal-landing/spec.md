# Модуль 2. Главная страница информационного портала

## Статус: Реализован

## Описание
Главная страница портала rosatom-travel.ru — точка входа с навигацией по всем разделам платформы.

## Реализованные секции

### 2.1 Hero-секция
- Градиентный баннер с заголовком «Гостеприимные города Росатома»
- Кнопки навигации: «Выбрать тур», «Города»
- Фоновое изображение с наложением

### 2.2 Блок «Что получает город»
- Сетка 3×2 карточек преимуществ вступления в программу
- Каждая карточка: фоновое изображение (Tilda CDN), градиентный оверлей, текст поверх
- Hover-эффект масштабирования изображения и подъёма карточки
- Анимация появления через scroll-reveal

### 2.3 Дополнительные инициативы
- Сетка 4×2 карточек (8 штук) на светлом фоне
- Каждая карточка: портретное изображение (Tilda CDN) + подпись
- Инициативы: горнолыжные базы, туристические клубы, автомобильный/водный/гастротуризм, АТОМотели, сувенирная линейка, мастер-планы городов
- Hover-эффект масштабирования и подъёма карточки
- Scroll-reveal анимация

### 2.4 Статистика
- Динамические счётчики: количество городов, туров, событий
- Данные из БД (не захардкожены)

### 2.3 Популярные туры
- Карточки featured-туров (до 6 штук)
- Изображение, название, город, цена

### 2.4 Города
- Карточки активных городов (до 6)
- Ссылки на детальные страницы

### 2.5 Интерактивная карта
- SVG-визуализация географии проекта
- Маркеры городов с позиционированием по координатам (lat/lng)
- Клик → переход на страницу города
- Отметка избранных для авторизованных

### 2.6 Timeline-лента событий
- Вертикальный таймлайн с чередованием карточек
- Типы: новости (синий), события (зелёный), вехи (янтарный)
- Дата, заголовок, описание, ссылка

### 2.7 Форма обратной связи «Хочу узнать подробнее»
- Поля: имя, email, телефон, сообщение
- Отправка через Inertia POST на `/contact`
- Сохранение в таблицу `contact_submissions`
- Flash-сообщение об успехе

### 2.8 Этапы программы (`/mainpage`)
- Формат карточек: фото + текст + CTA-кнопка
- Этап 1 «Исследования туристического потенциала атомных городов» → переход на страницу исследований (`research.index`)
- Этап 2 «Развитие гостеприимной инфраструктуры» → карточка с текстом (кнопка в состоянии «Скоро»)
- Этап 3 «Повышение компетенций муниципалитетов» → карточка с текстом (кнопка в состоянии «Скоро»)
- Этап 4 «Туры возможностей в атомные города» → переход на страницу туров возможностей (`opportunity-tours.index`)

### 2.9 Блок «Видеоролики» (`/mainpage`)
- Горизонтальный слайдер (snap scroll) видеокарточек
- Каждая карточка: превью-изображение с кнопкой Play, заголовок
- Клик → модальное окно с воспроизведением (embed iframe или `<video>`)
- Стрелки навигации по бокам на десктопе (при >3 видео)
- Данные захардкожены в компоненте (массив `videoItems`)

### 2.10 Блок «Новости» (`/mainpage`)
- До 3-х последних опубликованных записей из модели `Post`
- Карточки: изображение, дата, заголовок, excerpt
- Ссылка на полную статью (`blog.show`)
- Кнопка «Все новости» → переход на `blog.index`

### 2.11 Блок «Переезжаем» (`/mainpage`)
- CTA-баннер с градиентным фоном
- Кнопка «Подробнее» → переход на `/moving` (`moving`)
- Страница `/moving` — заглушка «Страница в разработке»

## Маршруты
- `GET /` → `HomeController@index` (публичный)
- `GET /mainpage` → `HomeController@mainpage` (публичная копия главной для отдельных правок)
- `GET /moving` → `HomeController@moving` (публичный, заглушка «в разработке»)
- `POST /contact` → `HomeController@contactSubmit` (публичный)

## Модели данных
- `TimelineEvent` — события для таймлайна
- `ContactSubmission` — обращения через форму
