# Блок «Город в объективе "Энергии городов"» + переключатели видимости блоков

## Цель

Добавить на публичную страницу города секцию «Город в объективе "Энергии городов"» с видео и описательным текстом, управляемую из админки. Добавить переключатели отображения блоков на странице города (в редакторе), чтобы админ мог скрывать/показывать секции.

## In-scope

### Фаза 1 (выполнена): блок «Энергия городов»
- Новые поля модели City (JSON `energy_cities_block`): `video_url`, `video_title`, `video_subtitle`, `description` (rich-text), `button_text`, `button_url`
- Миграция: добавить колонку `energy_cities_block` (JSON nullable) в таблицу `cities`
- Accessor в модели City для безопасного чтения полей блока
- Валидация в `Admin\CityController` (`store`, `update`) для вложенных полей `energy_cities_block.*`
- Админ-форма (`Admin/Cities/Form.vue`): секция «Энергия городов» с полями: video URL, заголовок, подзаголовок, описание (RichTextEditor), текст кнопки, URL кнопки
- Публичная страница (`Cities/Show.vue`): секция с двухколоночным layout по прототипу
- Тёмно-синий фон секции (#003274) с декоративными элементами согласно прототипу

### Фаза 2: переключатели видимости блоков
- Новое JSON поле `block_visibility` в таблице `cities` — хранит объект вида `{ facts: true, infrastructure: false, ... }`
- Переключатели (RCheckbox) в админ-форме для блоков:
  - Факты о городе (`facts`)
  - Инфраструктура (`infrastructure`)
  - Видео (`video`)
  - Достопримечательности (`attractions`)
  - Социальная сфера (`social_objects`)
  - Город в объективе «Энергии городов» (`energy_cities_block`)
- Публичная страница: каждый блок отображается только если его переключатель `true` **и** есть данные
- По умолчанию все блоки **включены** (true), чтобы существующие города не потеряли контент

## Out-of-scope

- Загрузка видеофайлов (только ссылки YouTube/RuTube)
- Отдельная таблица для блока (данные хранятся в JSON колонке модели City)
- Переключатели для блоков Галерея, Рецепты, Вакансии, Туры (не запрошены)

## Constraints

- Переиспользовать `RichTextEditor` для описания
- Переиспользовать логику embed из существующего `videoEmbedSrc` computed в `Show.vue`
- Максимум 5 файлов за шаг, до ~150 строк diff
- Переиспользовать `RCheckbox` из `@rosatom-ggr/ui-kit` (глобально зарегистрирован)
- При `block_visibility = null` (старые записи) считать все блоки включёнными
