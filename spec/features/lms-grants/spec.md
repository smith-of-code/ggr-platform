# Фича: LMS — Возможности

**Статус**: implemented

## Описание

Раздел «Возможности» показывает участникам гранты, субсидии, кредиты и события, а в админке позволяет управлять карточками возможностей и видеть участников, нажавших `Участвую`.

## Связанные сущности

- `App\Models\Lms\LmsGrant`
- `App\Models\Lms\LmsGrantDocument`
- `App\Models\Lms\LmsGrantEnrollment`
- `App\Models\Lms\LmsProfile`

## Админка

- Страница `/lms-admin/{event}/grants` показывает список возможностей, тип, период приёма заявок, количество участников и статус активности.
- Количество участников в списке открывает отдельную страницу `/lms-admin/{event}/grants/{grant}/participants` в новой вкладке.
- На странице участников показываются ФИО, email, телефон, проект (`project_description`), город, должность, организация и дата участия. Таблица поддерживает пагинацию.
- Кнопка `Выгрузить Excel` вызывает `Admin\GrantController@export`. На странице списка выгружаются все возможности, на странице участников — текущая возможность.

## Выгрузка

XLSX-отчёт содержит два листа: `Сводка` и `Участники`. В файле есть стилизованные заголовки, закреплённая шапка, ширины колонок, фильтры и перенос длинного текста. Отчёт содержит:

- название возможности;
- тип;
- период приёма заявок;
- статус приёма;
- ФИО участника;
- email;
- телефон;
- проект;
- город;
- должность;
- организация;
- дата участия.

Если у возможности нет участников, она всё равно попадает в отчёт одной строкой с пустыми полями участника.
