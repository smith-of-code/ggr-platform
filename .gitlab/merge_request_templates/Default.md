## Definition of Done

### Код
- [ ] Код соответствует принципам SOLID и KISS
- [ ] Соблюдены best practices для Laravel/Vue
- [ ] Нет закомментированного или неиспользуемого кода
- [ ] Нет `console.log()` и `dd()` в production коде
- [ ] Code style проверен (`php artisan lint`, `npm run lint`)
- [ ] Код прошел self-review

### Тесты
- [ ] Добавлены unit тесты (для Services/Models)
- [ ] Добавлены feature тесты (для API endpoints)
- [ ] Тесты проходят локально (`php artisan test`)
- [ ] Покрытие критичных путей достаточное
- [ ] N/A — тесты не требуются для этого типа изменений

### Документация
- [ ] Обновлены релевантные файлы в `/spec` (если применимо)
- [ ] Обновлены файлы в `/docs` (если user-facing feature)
- [ ] Добавлены PHPDoc/JSDoc комментарии
- [ ] Обновлен README.md (если нужно)
- [ ] Проставлены даты актуализации в спецификациях

### Миграции и обратная совместимость
- [ ] Database migrations протестированы
- [ ] Миграции имеют корректный `down()` метод
- [ ] Нет breaking changes (или описан migration path)
- [ ] Новые обязательные поля имеют default значения
- [ ] N/A — миграции не требуются

### Observability
- [ ] Добавлено логирование критичных операций
- [ ] Используются правильные log levels
- [ ] Нет чувствительных данных в логах (пароли, токены)
- [ ] Добавлены DB индексы для новых запросов (если нужно)
- [ ] Нет N+1 queries (используется eager loading)
